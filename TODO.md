- [x] 상행 종점(구간) 등록

  - [x] [예외] 역 사이에 새로운 역을 등록할 경우 기존 역 사이 길이보다 크거나 같으면 등록을 할 수 없음

- [x] 상행역과 하행역 둘 중 하나도 포함되어있지 않으면 추가할 수 없음
